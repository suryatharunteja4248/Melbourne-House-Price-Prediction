# 🏠 Melbourne House Price Prediction

## 📌 Overview
This project predicts house prices in Melbourne using a Linear Regression model with a full preprocessing pipeline.

## ⚙️ Preprocessing
- Missing value imputation (median & most frequent)
- One-hot encoding for categorical features
- Feature scaling using StandardScaler

## 🤖 Model
Linear Regression implemented using Scikit-learn Pipeline.

## 📊 Evaluation
- Actual vs Predicted visualization
- Residual analysis
- R² Score

## 🛠 Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
